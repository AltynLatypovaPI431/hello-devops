# Проект для практики DevOps

## Краткое описание
Проект создан для практического освоения DevOps-практик, в частности работы с Git, ветвлением, слиянием и разрешением конфликтов.

**Основные цели:**
- Освоение Git 
- Практика разрешения конфликтов слияния
- Работа с Pull Request

## Требования к среде
| Компонент | Версия |
|-----------|--------|
| Node.js   | 18.x   |
| npm       | 9.x    |
| Git       | 2.x    |

## Установка и запуск

### Клонирование репозитория

bash
```git clone https://github.com/AltynLatypovaPI431/1pr.git```
```cd 1pr```

\`\`\`bash
npm install
npm start
\`\`\`

## Тестирование
1. Запустите тесты: \`npm test\`
2. Проверьте линтер: \`npm run lint\`
3. Проверьте покрытие кода: \`npm run coverage\`
